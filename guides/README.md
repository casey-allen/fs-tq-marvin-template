# MARVIN Setup Guides

Interactive walkthroughs to get the most out of MARVIN. Each guide can be read on GitHub or run interactively inside MARVIN with `/guide`.

## Quick Start

Inside MARVIN, type:
- `/guide` — see all guides with completion status
- `/guide getting-started` — start a specific guide

## Available Guides

| # | Guide | What It Covers | Time |
|---|-------|---------------|------|
| 01 | [Getting Started](01-getting-started.md) | Install Claude Code, clone template, run setup | 15 min |
| 02 | [Skills & MCP](02-skills-and-mcp.md) | Codex marketplace, company skills, MCP integrations | 10 min |
| 03 | [Obsidian Vault](03-obsidian-vault.md) | Git-tracked second brain with daily notes and task tracking | 20 min |
| 04 | [Team Digest](04-team-digest.md) | Daily Jira/Slack/GitHub synthesis posted to your team's Slack | 15 min |
| 05 | [Weekly Review](05-weekly-review.md) | Weekly planning with Obsidian notes + Jira project pulse | 15 min |
| 06 | [Keeping Updated](06-keeping-updated.md) | /sync, Codex marketplace updates, blueprint regeneration | 5 min |

## Recommended Order

Start with **01 Getting Started** (required). After that, we recommend:
1. **02 Skills & MCP** — get company-wide skills like NRFC writer
2. **03 Obsidian Vault** — set up your second brain (required for 05)
3. **04 Team Digest** — daily engineering intelligence for your team
4. **05 Weekly Review** — automated weekly planning (requires 03)
5. **06 Keeping Updated** — learn how to stay current

Guides 04 and 05 are independent of each other — do them in either order.

## Prerequisites

- GitHub SSH or HTTPS access to `fluidstackio/fs-tq-marvin-template`
- macOS or Linux
- A terminal you're comfortable with (guides explain every command)
